# Ujwal Ojaswi — Financial Analyst Portfolio

A personal portfolio site positioned for Financial Analyst roles: financial modeling, valuation, risk analysis, and reporting. Built with React, Vite, TypeScript, Tailwind CSS v4, and Framer Motion.

This is a sibling site to [the Data Analyst portfolio](https://portfolio-liard-two-62.vercel.app) — same person, same underlying experience, reframed and re-curated for finance-focused roles.

## Stack

React + Vite + TypeScript, Tailwind CSS v4 (`@tailwindcss/vite`, CSS-first `@theme`), Framer Motion, React Router.

## Development

```
npm install
npm run dev
npm run build
```
